# Customer Churn Prediction (ML Project)

## Overview
This project is an end-to-end machine learning pipeline for predicting customer churn using the Telco dataset.

## Problem
Classification task: predict whether a customer will leave (Churn = Yes/No)

## Dataset
- 7043 customers
- 21 features (demographics, services, billing)

## Pipeline
- Week 1: Problem definition & hypothesis
- Week 2: Data preprocessing & EDA
- Week 3: Model building (Logistic Regression, KNN)
- Week 4: Model evaluation

## Key Features
- Data cleaning & preprocessing
- Feature engineering
- Exploratory data analysis (EDA)
- Machine learning models

## Tech Stack
- Python
- Pandas, NumPy
- Matplotlib
- Scikit-learn

## Results
Identified key churn drivers:
- Contract type
- Monthly charges
- Tenure
